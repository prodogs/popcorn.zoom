# Popcorn.zoom.js


Zoom and Rotate Popcorn video objects, like this...

	var $pop = Popcorn( "#video-id" );

	//	zoom to 1.5 the original size
	$pop.zoom( 1.5 );

	//	zoom to 1.5 the original size and rotate 45 degrees
	$pop.zoom( 1.5, 45 );
